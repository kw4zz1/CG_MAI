# Компьютерная графика — лабораторные работы (2025/2026)

Короткое описание
------------------
Это рабочее дерево с решениями лабораторных работ по курсу «Компьютерная графика» (5 семестр, 3 курс 8 института). Проект использует Vulkan + небольшой учебный каркас `veekay` для организации рендеринга, шейдеров и UI (Dear ImGui). В репозитории находятся несколько подпроектов/папок с отдельными лабораторными и примерами.


Структура репозитория
----------------------
- `veekay_lab1/`  — ЛР 1: генерация геометрии (сфера/куб), модель‑вид‑проекция, push‑constants, per-vertex color.
- `veekay_lab23/` — ЛР 2/3: освещение (Blinn‑Phong), текстурирование, материалы, UI.
- `veekay_lab234/` — ЛР 2/3/4: расширенные примеры (камера, несколько типов света, shadow mapping и dynamic rendering).

Требования
-----------
- Linux (или другая система с работоспособным Vulkan драйвером)
- CMake >= 3.18
- Ninja или Make
- Vulkan SDK (для сборки шейдеров и разработки)
- Установленные зависимости разработки (GLM/other headers, в проекте часть зависимостей поставлена как subproject)

Быстрая сборка и запуск (Linux)
--------------------------------
1. Создайте директорию сборки и вызовите CMake:

```bash
cd /veekay_lab..
cmake --preset debug
cmake --build build-debug --parallel
```

2. Запуск примера (пример для тестового приложения):

```bash
./build-debug/testbed/testbed
```

Примечания по сборке
--------------------
- Проект использует CMake Presets и добавляет несколько внешних зависимостей поддиректориями (`_deps/`). При первом запуске сборки CMake скачает или подготовит субмодули/подпроектов.
- Шейдеры компилируются через CMake (glsl → SPIR‑V). При необходимости можно компилировать шейдеры вручную через `glslangValidator`.
